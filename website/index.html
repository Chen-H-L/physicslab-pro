面向教学场景设计

把抽象规律变成可观察、可调整、可分析的实验过程。

平台围绕“仿真演示、图样分析、数据处理、智能答疑”组织工作流，适合教师课堂演示、 学生实验预习、微课素材制作和课程项目展示。界面保留清晰的参数控制、图像反馈和结果输出， 帮助使用者从现象一路走到模型和结论。

核心能力

一套工具覆盖完整实验教学链路

01

光学虚拟仿真

支持牛顿环、劈尖干涉、双缝干涉，参数调整后实时生成干涉图样与实验模型示意。

02

光学图样分析

从图片或视频中提取亮度分布，完成峰值统计、条纹间距分析和结果导出。

03

热力学模拟

展示理想气体分子运动、活塞变化和 P-V-T 图像联动，便于讲解状态过程。

04

振动学实验室

覆盖简谐振动、同方向合成、拍振和正交合成，支持 GIF 微课素材导出。

05

数据工作台

提供数据录入、统计分析、自动拟合与不确定度计算，承接实验结果处理。

AI

虚拟助教

兼容 OpenAI Chat Completions 风格接口，可结合实验现象、曲线和问题进行答疑。

真实界面

从演示到分析，界面围绕课堂工作流展开

每个页面都保留参数、图像、曲线和结果区域，让教师能边调节边讲解，学生能边观察边验证。

课堂使用路径

从提出问题到形成结论，只保留必要步骤

  1. 设定实验

    选择实验类型，调整波长、缝距、体积、频率或数据输入方式。

  2. 观察现象

    实时查看干涉图样、分子运动、振动轨迹和动态图表。

  3. 分析数据

    提取曲线、统计峰值、拟合模型并计算实验不确定度。

  4. 组织表达

    结合 AI 助教整理现象解释、公式关系和实验结论。

技术实现

稳定的桌面端架构，服务高频教学操作

项目以 PyQt6 作为界面底座，结合 NumPy、SciPy、OpenCV、Matplotlib 与 Pandas 完成模拟、 图像处理和数据分析，并通过后台线程处理视频分析与 AI 请求。

  • PyQt6 桌面界面
  • OpenCV 图像与视频处理
  • NumPy / SciPy 科学计算
  • Matplotlib 嵌入式图表
  • Pandas 数据表处理
  • OpenAI SDK 兼容接口
